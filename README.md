[//]: # ( Description:
WP Multilanguage is a plugin that allows you to translate your WordPress website into multiple languages.


[//]: # ( Installation:
1. Download the plugin from [GitHub](
2. Upload the plugin to the `/wp-content/plugins/` directory
3. Activate the plugin in the `Plugins` menu in WordPress
4. Go to the `Settings` menu in the `Multilanguage` section and add the languages you want to use
